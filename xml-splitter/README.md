# XML/HTML Splitter Web App

A simple web tool to split large XML/HTML files into smaller chunks based on headings or selectors.

## Running Locally

1. `npm install`
2. `npm start`
3. Open `http://localhost:3000`

## GitHub Codespaces Info

This project is configured for GitHub Codespaces.

**Free Tier Note:**
- GitHub Codespaces Free Tier includes **120 core hours** per month and **15GB** of storage.
- To conserve usage:
  1. Go to your GitHub Settings > Codespaces.
  2. Set "Default idle timeout" to **30 minutes** (or less) to prevent burning hours if you leave the tab open.
  3. Delete the codespace when you are finished with your task to free up storage.

**Storage Note:**
- The app automatically cleans up uploaded files after processing to keep disk usage low.
